## рџ“Њ Project Overview

The **Kawasaki Robot Tic-Tac-Toe** is an interactive robotics project developed for a technical event, where an industrial Kawasaki robotic arm competes against a human player in a game of Tic-Tac-Toe.

Although the game concept was not originally developed by me, I was responsible for the **complete project development, hardware integration, robot programming, system setup, testing, and event deployment**. This project gave me valuable hands-on experience in industrial robotics, embedded systems, automation, and multidisciplinary project coordination.

The system combines a **Kawasaki Industrial Robot**, **Arduino Mega**, **magnetic sensors**, and **DMX-controlled LED lighting** to create an engaging and reliable interactive experience for visitors during the event.
